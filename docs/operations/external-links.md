# External Links

## 管理方針

外部リンクは、ユーザー導線と運用上の変更頻度が高いため、この文書にも記録します。リンクを変更した場合は、該当コンポーネントとこの文書を同時に更新してください。

## リンク属性

外部リンクには原則として以下を付与します。

```tsx
rel = 'noopener noreferrer';
target = '_blank';
```

紹介・広告性のあるリンクは、必要に応じて `nofollow` も検討します。
