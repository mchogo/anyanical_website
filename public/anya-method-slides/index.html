01 / BASIC CONCEPT

基本の考え方 — ダウ理論がベース

上昇トレンドなら「前の高値は基本的に抜けてくる」。調整終わりから前の高値までを取る。

上昇トレンド（環境認識が上）なら、前の高値は基本的に抜けてくる
調整が入って終わる形になったら、前の高値まで取るイメージ
そこに SMC / ICT の概念を組み合わせて優位性を高め、分類・体系化したものがアニャニカル
02 / MULTI TIME FRAME

MTF — 3つの時間足に役割を割り当てる

基準は ICT Timeframe Alignment。どの組み合わせでもやることは同じ（フラクタル構造）。

執行足（表示足）監視足環境認識足
1分足15分足4時間足
5分足1時間足日足
15分足4時間足週足
1時間足日足月足
銘柄も時間軸も考え方は同じ。性格・生活スタイルに合わせて選定すればOK
03 / TERMS ①

実体抜け — 「終値」で抜けて初めて成立

定義：上げ止まり／下げ止まりの基準足の「有効な」高安を終値で抜けること。

ヒゲだけラインを抜けても、終値が抜けずに確定 → 実体抜けではない
終値がラインの外で確定 → 実体抜け成立
1本の足の内部で動いているだけの足は無視。内部から抜け出さない限り波に影響しない
04 / TERMS ②

安値切り上げ — 3条件で判定

エントリー時に一番見ることになる判定。高値切り下げは反対に考えるだけ。

1

対象の波（パス）の半値以上を戻している

2

前の安値を割らずに下げ止まっている

3

実体で高値を超える足が出ている（＝実体抜けと同じ判定）

戻さないまま行ってしまったら「ご縁がなかった」と切り替えて次へ
05 / TERMS ③

高安の切り上げ／切り下げ — 両方を見る

安値切り上げ＋高値の実体抜けの両方が揃った状態。パターン③の判定で使う。

安値切り上げが確定している
＋ 直近高値も実体で抜けている → 次の高値は必然的に上
高値だけ切り下げ・安値は切り下げていない…など片方だけでは呼ばない

まずAIの列を見る。次に環境認識足の状態（下げ止まり／上昇中）を必ず確認。

ロング狙いなら、環境認識足が下げ止まっているか（テーブルで「上昇中」か）を確認
形成中の足でラインを抜いていても確定までは「下落中」表示のまま。「調整ライン抜け」表示が出たら要注意
07 / STEP 2

事前作業 — 監視足の「調整」を半値まで待つ

ICT/SMCの Premium / Discount がベース。大きい波に対する半値以上の調整終わりが優位。

ロングなら、一度監視足が上げ止まる（調整が入る）のを待つ
半値以上戻している調整の方が優位性が高い
小さい波の半値より、大きい波に対する半値以上の調整終わりを狙う方が優位
08 / ENTRY PATTERN

パターン① — 監視足が下げ止まっている時（基本形）

AI環境認識：上 ／ 環境認識足：上 ／ 監視足：上 → 執行足の調整終わりからエントリー。

SL1：RRは最高だが刈られやすい。刈られても入り直せる人向け
SL2：「監視足が下げ止まった」根拠が崩れたら損切り（筆者の基本）
SL3：監視足のダウが崩れたら損切り
すでにターゲット到達済みなら次の調整を待つ。「どこを狙っているか」を常に明確に
09 / ENTRY PATTERN

パターン② — 監視足がまだ調整中の時（応用）

条件：監視足のパスが「高安切り下げ」ではない → 執行足の安値切り上げからIN。

執行足の安値切り上げ → そのあと監視足が下げ止まってくる可能性が高い（下位足は上位足に影響を及ぼせる）
SL：安値切り上げが否定される位置
ショートなら読み替えて高値切り下げから
10 / ENTRY PATTERN

パターン③ — 監視足が「修正」している時（最難・最高RR）

監視足パスが高安切り下げ中 → 執行足で高安の切り上げ（両方）を待つ。

1

執行足で高値の切り上げを確認（②と違い高値側も必須）

2

実体抜けの足で安値の切り上げも確定 → エントリー可

SLは前の安値の下。TPは他パターンと同様
逆三尊・ダウ転換・ICTの2022モデル…各理論の買いが重なる形だから優位。抵抗と組み合わせ＋検証を重ねる
11 / SUPPLEMENT

補足編 — 半値戻さない時・抵抗・建値

前提：「保存版の内容だけで勝てる」。基本を落とし込んだ上での追加要素。

📍

半値戻さない時 → 節目・抵抗

「買われやすい／売られやすい」ポイントなら条件成立でIN可。節目＝前日・前週・前月の高安／始値。特に上げ下げ止まりライン・圧力ラインは攻防が起きやすい。

🚫

脳死の逆張りはNG

環境認識が上なのに「レジ来たからショート！」はやらない。抵抗はロングの決済に使うのが正解。攻防が起きるかは「想定」であって確定ではない。いつものパターンを待つ。

🛡️

建値のタイミング

①ダウ理論：前回高値を実体で超えたら（そこは割ってこないはず）②圧力ラインが切り上がったら。トレンド中＆最低TP未到達で活用。レンジで建値は無限に狩られる。

建値を入れるか自体はメンタル・生活スタイルとの相談。刈られて暴走する型なら入れない、損で頭が熱くなる型なら入れる。
